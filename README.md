# Setup do Projeto Livraria Django

Este arquivo contém todas as instruções necessárias para instalar, configurar e rodar o projeto em um único lugar.

## Requisitos

- Python 3.9+
- pip
- npm (instale o Node.js que já inclui o npm)
- Django 3.2+
- Node.js 14+
